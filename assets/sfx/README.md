# Custom sound effects

Drop mp3 files here to replace the built-in synth sounds. The game picks up
whatever exists; anything missing keeps its synth version. No code changes or
manifest needed — just the exact filenames below.

| file | plays when |
|---|---|
| `quack.mp3` | bouncing on the rubber duck |
| `jet.mp3` | jet flight (loops for the whole flight — use a seamless loop) |
| `jump.mp3` | landing on a platform |
| `spring.mp3` | hitting a spring |
| `coin.mp3` | collecting a coin |
| `power.mp3` | picking up the jet powerup |
| `shield.mp3` | shield pickup / shield break |
| `hit.mp3` | dying |
| `zap.mp3` | smashing a drone/missile during flight |
| `over.mp3` | game over jingle |
| `click.mp3` | UI button click |
| `alarm.mp3` | missile warning |
